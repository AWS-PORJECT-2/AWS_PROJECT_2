# 두띵 이미지 에셋 — 생성 프롬프트 모음

> 각 항목의 프롬프트는 **공통 스타일이 이미 합쳐진 완성본**이라 그대로 복붙하면 됩니다.
> 생성한 파일은 `frontend/assets/<파일명>` 에 그 이름 그대로 넣으면 사이트에 자동 적용됩니다.
> 권장: 배경 투명(PNG), 정사각(아이콘) / 명시된 비율(일러스트·배너).

공통 스타일 (참고용 — 아래 프롬프트엔 이미 포함됨):
`flat vector illustration, soft rounded shapes, minimal clean style, light purple (#8B5CF6) accent on white background, consistent line weight, no text, no emoji, high resolution, centered, generous padding`

3. 알려주면 내가 배포에 반영(S3 업로드)하거나, 직접 넣었으면 그대로 사이트에 보임.

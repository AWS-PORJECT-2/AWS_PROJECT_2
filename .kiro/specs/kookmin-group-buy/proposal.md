# 국민대학교 공동구매 · 중고거래 통합 플랫폼

## 프로젝트 소개

국민대학교 학생들은 과잠, 후리스, 롱패딩, 버스 대절 등 다양한 공동구매를 진행하고 있다.
하지만 현재는 구글 폼으로 수기 관리하고 있어서, 참여 현황 파악이 어렵고 마감 처리도 번거롭다.

이 프로젝트는 **공동구매 전용 웹 플랫폼**을 만들어 이 문제를 해결하고,
플랫폼 활용도를 높이기 위해 **교내 중고거래 기능**도 함께 제공한다.

학교 메일(@kookmin.ac.kr) 인증을 통해 국민대 재학생만 이용할 수 있는 안전한 환경을 보장한다.

---

## 기존 문제점

- 구글 폼으로 공동구매를 진행하면 참여 현황을 실시간으로 확인하기 어렵다
- 옵션(사이즈, 색상 등) 관리가 복잡하고 집계에 시간이 많이 걸린다
- 마감 처리, 최소 인원 확인 등을 개설자가 수동으로 해야 한다
- 교내 중고거래는 에브리타임 등 외부 플랫폼에 의존하고 있다

---

## 해결 방안

### 공동구매

- 게시글 하나로 제목, 설명, 가격, 옵션, 마감일, 최소 인원을 한번에 설정
- 참여자가 옵션과 수량을 선택하면 참여 현황이 자동으로 집계됨
- 마감일이 되면 최소 인원 충족 여부에 따라 자동으로 성사/무산 처리
- 개설자는 참여자 목록(이름, 학번, 옵션, 수량)을 바로 확인 가능
- AI가 상품명과 키워드만 입력하면 게시글 초안을 자동 생성해줌

### 중고거래

- 교재, 전자기기, 의류 등 카테고리별로 중고 물품 등록
- 판매중 → 예약중 → 거래완료 상태 관리
- 쪽지 기능으로 구매자와 판매자가 직접 소통

### 인증

- @kookmin.ac.kr 메일로만 가입 가능
- 이메일 인증 후 학번과 이름을 등록하여 재학생 확인

---

## 주요 기능 요약

| 영역 | 기능 |
|------|------|
| 인증 | 학교 메일 회원가입, 로그인/로그아웃, 비밀번호 변경 |
| 공동구매 | 게시글 작성, 참여/취소, 현황 확인, 자동 마감, AI 글 생성 |
| 중고거래 | 물품 등록, 검색/카테고리 필터, 상태 관리, 쪽지 |
| 공통 | 알림(상태 변경, 새 쪽지), 마이페이지(내 활동 내역) |

---

## 대상 사용자

- 국민대학교 재학생

## 플랫폼

- 반응형 웹 (모바일 + 데스크톱)

---

## 기대 효과

- 공동구매 개설부터 마감까지 하나의 플랫폼에서 자동화
- 참여 현황, 옵션 집계를 실시간으로 확인하여 운영 부담 감소
- 교내 학생 간 중고거래를 안전하고 편리하게 진행
- AI 활용으로 게시글 작성 시간 절약

📌 프로젝트 소개

국민대 학생들은 과잠, 후리스, 롱패딩, 버스 대절 등
다양한 공동구매를 진행하고 있지만,
현재 구글 폼으로 수기 관리하고 있어 비효율적입니다.

이 프로젝트는 공동구매 전용 웹 플랫폼을 만들어 이 문제를 해결하고,
교내 중고거래 기능도 함께 제공합니다.

학교 메일(@kookmin.ac.kr) 인증으로 재학생만 이용 가능합니다.

──────────────────────────────────────────

❗ 기존 문제점

• 구글 폼으로는 참여 현황을 실시간 확인이 어려움
• 옵션(사이즈, 색상) 관리가 복잡하고 집계에 시간 소요
• 마감 처리, 최소 인원 확인을 개설자가 수동으로 해야 함
• 교내 중고거래는 외부 플랫폼에 의존 중

──────────────────────────────────────────

✅ 해결 방안

[공동구매]
• 게시글 하나로 제목/설명/가격/옵션/마감일/최소인원 설정
• 참여자가 옵션·수량 선택하면 현황 자동 집계
• 마감일에 최소 인원 충족 여부 따라 자동 성사/무산 처리
• 개설자는 참여자 목록(이름, 학번, 옵션, 수량) 바로 확인
• AI가 상품명+키워드로 게시글 초안 자동 생성

[중고거래]
• 교재/전자기기/의류 등 카테고리별 물품 등록
• 판매중 → 예약중 → 거래완료 상태 관리
• 쪽지 기능으로 구매자-판매자 직접 소통

[인증]
• @kookmin.ac.kr 메일로만 가입 가능
• 이메일 인증 후 학번/이름 등록으로 재학생 확인

──────────────────────────────────────────

📋 주요 기능 요약

  인증      학교 메일 회원가입, 로그인/로그아웃, 비밀번호 변경
  공동구매  게시글 작성, 참여/취소, 현황 확인, 자동 마감, AI 글 생성
  중고거래  물품 등록, 검색/카테고리 필터, 상태 관리, 쪽지
  공통      알림(상태 변경, 새 쪽지), 마이페이지(내 활동 내역)

──────────────────────────────────────────

👥 대상 사용자  국민대학교 재학생
🌐 플랫폼      반응형 웹 (모바일 + 데스크톱)

──────────────────────────────────────────

🎯 기대 효과

• 공동구매 개설~마감까지 하나의 플랫폼에서 자동화
• 참여 현황·옵션 집계를 실시간 확인 → 운영 부담 감소
• 교내 학생 간 중고거래를 안전하고 편리하게 진행
• AI 활용으로 게시글 작성 시간 절약
